Malaysia Tree Cover Loss Dashboard (2001–2024)

Annual Tree Cover Loss by State (2017–2024)

This line chart visualises the annual tree cover loss in five Malaysian states — Sarawak, Sabah, Pahang, Kelantan, and Johor — from 2017 to 2024. Each line represents one state, showing how deforestation rates have fluctuated over time. The chart highlights key years where forest loss spiked or decreased, helping identify regional patterns and potential causes such as agricultural expansion or policy interventions.

Tree Cover Loss in 2024 by State

This bar chart compares the amount of tree cover loss in 2024 across all Malaysian states. It provides an at-a-glance view of which regions have experienced the highest recent deforestation levels. The data can be used to prioritise reforestation efforts and monitor environmental policy effectiveness.
